# extras-skills

Non-trading, non-financial [Agent Skills](https://agentskills.io/specification) for Longbridge. Kept separate from [longbridge/skills](https://github.com/longbridge/skills), which covers securities/trading functionality.

## What's inside

| Skill                                            | Description                                                                                 |
| ------------------------------------------------ | --------------------------------------------------------------------------------------------- |
| [longbridge-lbti](./skills/longbridge-lbti) | 長橋 LBTI 投資風格測試 — a 5-question quiz that sorts users into 8 investing-personality types, returning a result card + QR code. |

## Install

Same install paths as `longbridge/skills` — see that repo's [docs/install.md](https://github.com/longbridge/skills/blob/main/docs/install.md) for details (npx / bun / Claude Code marketplace / clone+symlink).
